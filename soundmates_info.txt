Η εφαρμογή μας έχει την εξής αρχικτεκτονική:

[soundmates app] <-> [soundmates api at http://soundmates.ddns.net:5000] <-> [mysql database at localhost]

Δηλαδή όλα τα δεδομένα της αποθηκεύονται στην βάση μας στο pc μας και οι εικόνες, ηχητικά μηνύματα σε κάποια local directories.

Για λόγους ασφάλειας, καθώς δεν έχουμε κάποιο αμυντικό μηχανισμό προς κακόβουλους χρήστες του api, ο server δεν θα παραμένει συνεχώς ανοιχτός. Σίγουρα όμως θα είναι ανοιχτός την ώρα της εξέτασης. 
Σε περίπτωση που θέλετε να δοκιμάσετε την εφαρμογή πριν ή μετά την εξέταση, μπορείτε να στείλετε ενα mail εδώ: kkatgm@gmail.com.
Για την ομαλή επίδειξη της εφαρμογής έχουμε φτιάξει κάποια dummy profiles μέσω κώδικα(που έχουν τις ελάχιστες δυνατές προυποθέσεις χρήστη) αλλά και μερικά προφίλ με φωτογραφίες, κουτιά περιγραφής, κουτιά με φωτητικά μηνύματα και social με προτιμήση σε Rock μουσική και προτιμήση φύλου "any".
Άρα για να εμφανιστούν αυτά τα "ολοκληρωμένα" προφίλ δεν έχετε παρά να κάνετε select genres -> Rock με προτίμηση >= 50%.
Επιπλέον καταφέραμε να χρησιμοποιήσουμε το API του Spotify για να εξάγουμε τις προτιμήσεις του χρήστη σε μουσική, μέσω του λογαριασμού του. Δυστυχώς, η πολιτική του Spotify μας αφήνει να το χρησιμοποιήσουμε μόνο σε εγκεκριμένους λογαριασμούς μέχρι η εταιρεία να αξιολογήσει την εφαρμογή μας. Καθώς το αυτή η διαδικασία κρατάει μέχρι 6 εβδομάδες, δεν είμαστε σε θέση να προσφέρουμε αυτή τη λειτουργικότητα στο ευρύ κοινό.

Παρ' όλα αυτά ο κώδικας παραμένει σε comments στην εφαρμογή ώστε να μπορεί να εξεταστεί και θα χαρούμε να σας κάνουμε μια επίδειξη στην παρουσίαση της εργασίας αν κάτι τέτοιο σας ενδιαφέρει.
Μπορείτε μάλιστα να μας στείλετε το όνομα ενός account σας στο spotify για να το βάλουμε στην λίστα με τους developer λογαριασμούς και να το δοκιμάσετε και εσείς.
